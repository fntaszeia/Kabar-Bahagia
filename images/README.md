# Images Folder

Please add your wedding photos to this folder:

## Required Images:

1. **hero-bg.jpg** - Main background image for the hero section (1920x1080px recommended)
2. **bride.jpg** - Photo of the bride (500x500px minimum, square format)
3. **groom.jpg** - Photo of the groom (500x500px minimum, square format)
4. **photo1.jpg** - Gallery photo 1 (800x800px recommended)
5. **photo2.jpg** - Gallery photo 2 (800x800px recommended)
6. **photo3.jpg** - Gallery photo 3 (800x800px recommended)
7. **photo4.jpg** - Gallery photo 4 (800x800px recommended)

## Placeholder Images:

If you don't have images yet, you can use placeholder services:
- https://placeholder.com/
- https://picsum.photos/
- Or use any free stock photo sites like Unsplash or Pexels

## Tips:
- Use high-quality images
- Optimize images for web (compress them to reduce file size)
- Keep aspect ratios consistent for gallery photos
- Use square format for bride and groom photos
